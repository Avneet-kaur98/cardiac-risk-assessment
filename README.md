# Cardiac Risk Assessment

A tabular machine learning project that cleans and trains a heart-disease
risk model in Google Colab, compares three algorithms, adds explainability,
and serves live predictions through a Flask dashboard with an animated
risk gauge.

## Project Description

This project predicts a patient's estimated heart disease risk from 13
clinical vitals (age, chest pain type, cholesterol, blood pressure, and
others), using the Statlog Heart Disease dataset. Three models —
Logistic Regression, Random Forest, and Gradient Boosting — were trained
and compared using ROC-AUC, with Logistic Regression selected as the best
performer (87% accuracy, 0.914 ROC-AUC). The trained pipeline is served
through a Flask web app that auto-builds its input form from the model's
own metadata, shows an animated risk gauge, and explains each prediction
with a "top factors" breakdown showing which patient values most
influenced the result.

**Tech stack:** Python, scikit-learn, pandas, Flask, HTML/CSS/JavaScript
(Jinja templating), Google Colab.

## Project Structure

```
CardiacRiskAssessment/
├── Heart_Disease_Risk_Training.ipynb   ← Colab notebook: cleaning, training, explainability
├── requirements.txt                    ← Python dependencies
├── README.md                           ← this file
└── flask_app/
    ├── app.py                          ← Flask backend
    ├── requirements.txt
    ├── model/                          ← trained model files go here (see note below)
    │   ├── model_pipeline.pkl
    │   ├── feature_info.json
    │   └── insights.json
    ├── templates/
    │   └── index.html                  ← dashboard UI
    └── static/
```

**Note:** `model_pipeline.pkl`, `feature_info.json`, and `insights.json`
are generated by running the notebook and are not included in this
repository/zip by default since they're produced fresh each time the
notebook runs. Run `Heart_Disease_Risk_Training.ipynb` in Colab first,
then place the 3 downloaded files into `flask_app/model/` before running
the Flask app.

## How to Run

**1. Train the model (Google Colab):**
Upload `Heart_Disease_Risk_Training.ipynb` to Colab and run all cells.
Download the 3 generated files into `flask_app/model/`.

**2. Run the Flask app (locally):**
```bash
cd flask_app
pip install -r requirements.txt
python app.py
```
Open http://127.0.0.1:5000

**3. Or run it live from Colab directly** (no local install needed) —
see the in-notebook instructions for launching via Colab's built-in
`proxyPort` link.

## Limitations

This is an educational demo trained on a small (270-patient) public
dataset. It is not clinically validated and is not a substitute for
professional medical diagnosis.
